## Transactions Volume Plot Results

> **A folder for preserving The Results of ggplot 2 visualization from R Code.**

This folder preserves the result of transactions volume of all digital payment systems
operated in Indonesia, including Debit and Credit Cards and Electronic Money. 
All resutls are derived from the R code based on ggplot2 package visualization.
