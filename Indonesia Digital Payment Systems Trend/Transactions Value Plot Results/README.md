## Transactions Value Plot Results  

> **A folder for preserving The Result of ggplot2 visualization from R code.**

This folder preserves the result of transactions value of all digital payments operated in Indonesia, such as Debit and Credit Cards. All results are derived from the R code based on ggplot2 package visualization.
